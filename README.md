## 主要研究方向
我的研究方向主要研究方向是网络物理系统（CPS）的性能及可靠性研究。目前，集中在两个子方向：
- 面向服务的物联网中基于信任的数学建模及性能评估。
- 4G/5G车联网及智能交通中服务质量（QoS）相关的性能评估和可靠性研究。
